LangGraph Agent Web - Quick Start Guide
====================================

Welcome to LangGraph Agent Web!
This is an AI chat system based on DeepSeek with a friendly web interface.

====================================
FIRST TIME SETUP
====================================

1. Configure API Key
   - Create file: %USERPROFILE%\.lex_mint\keys_config.yaml
   - Put your key in:
     providers:
       deepseek:
         api_key: "your_key_here"

2. Install Dependencies (First time only)
   - Double-click install.bat
   - Wait for installation to complete (5-10 minutes)
   - Press any key to close when you see "Installation Complete"

3. Start Services
   - Double-click start.bat
   - Two command windows will open (backend and frontend)
   - Browser will automatically open http://localhost:<FRONTEND_PORT>

====================================
DAILY USAGE
====================================

Start:        Double-click start.bat
Stop:         Double-click stop.bat
Menu:         Double-click menu.bat (Recommended!)
Check Status: Double-click status.bat

====================================
FEATURES
====================================

* Web Chat Interface
  - Access: http://localhost:<FRONTEND_PORT>
  - Multi-turn conversations
  - Auto-save chat history

* Conversation History
  - Location: conversations folder
  - Format: Markdown files
  - Can be opened/edited with any text editor

* Session Management
  - Create new conversations
  - View chat history
  - Delete unwanted sessions

====================================
SYSTEM REQUIREMENTS
====================================

- Windows 10 or higher
- Python 3.10 or higher
- Node.js 18 or higher
- Stable internet connection

====================================
TROUBLESHOOTING
====================================

Q: Browser won't open after starting?
A: Wait about 10 seconds for services to start

Q: "Virtual environment not found" error?
A: Run install.bat first

Q: Cannot send messages?
A: Check if API key in %USERPROFILE%\.lex_mint\keys_config.yaml is correct

Q: How to backup conversations?
A: Copy the entire conversations folder

====================================
ACCESS URLs
====================================

Web Interface:  http://localhost:<FRONTEND_PORT>
API Backend:    http://localhost:<API_PORT>  (configurable in .env)
API Docs:       http://localhost:<API_PORT>/docs

Note: To change ports, edit API_PORT and FRONTEND_PORT in .env file
